════════════════════════════════════════════════════════════════════════════════
RATE LIMITING NOTES
════════════════════════════════════════════════════════════════════════════════

NVIDIA Proxy (port 8788):
- --max-rpm 30 keeps a safety buffer below the advertised 40 RPM per endpoint.
- --max-concurrency 1 prevents burst pairs from arriving at the provider at the same second.
- --max-input-bytes-per-minute 350000 throttles very large context bursts (~110KB per request).
- --max-wait-seconds 25 ensures requests don't queue > 25s (Copilot timeout ~90-120s).

MiniMax Proxy (port 8789):
- NO rate limiting (paid API, unlimited requests)
- --max-wait-seconds 120 is default for Copilot timeout safety (can be adjusted)
